Portfolio Website Preview

How to Run Your Portfolio Website

Your portfolio website is built with modern web technologies including:

  • React for the frontend
  • Vite as the build tool
  • TypeScript for type safety
  • Express.js for the backend
  • Tailwind CSS for styling

Setup Instructions

  1. Install Node.js (which includes npm) from nodejs.org
  2. Open a terminal/command prompt in your project folder
  3. Run npm install to install all dependencies
  4. Run npm run dev to start the development server
  5. Open your browser and navigate to http://localhost:5000 (or the port shown in the terminal)

Project Structure

Your portfolio website includes the following key sections:

  • Hero - Introduction and main banner
  • About - Information about you
  • Skills - Your technical and professional skills
  • Projects - Showcase of your work
  • Research - Your research papers and academic work

Alternative Preview Method

Since Node.js isn't properly set up, you can view a simple preview using Python's built-in HTTP server:

  1. Open a terminal/command prompt in your project folder
  2. Run: python -m http.server 8000
  3. Open your browser and navigate to http://localhost:8000/preview.html
  4. Note: This will only show this page, not your actual portfolio
